Introduction

Shen is a ninja who has a lot of problems with lord Zed, which have hijacked his granma. He have his objective clear, rescue his granma from Zed, going to his castle.

How to play

A - Left // D - Right // F - Dash // SPACE - Jump

DEBUG: F9 - View colliders // F10 - Godmode // F1 - First map // F2 - Second map // F3 - Restart

Innovations

We have done a new move that is a dash, this dash can be aerial or grounded, we have added also death sounds for each entity, and some kind of animations. It is possible to access the web from the game in credits
