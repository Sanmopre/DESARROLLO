Controls:
	-A = left player movement
	-D = right player movement
	-SPACE = player jumping	
	-F = player dash
	-E = player attack

	debug tools:
	F9 = view colliders
	F10 = God Mode
	F1 = change to first stage
	F2 = change to second stage
	F3 = restart current stage
